Начало работы

Как создать учетную запись?

Для создания учетной записи:

  1. Перейдите на страницу регистрации
  2. Заполните форму вашими данными
  3. Подтвердите email, перейдя по ссылке в письме
  4. Войдите в систему с вашими учетными данными

Как добавить сотрудников?

Администраторы могут добавлять сотрудников:

  1. Перейдите в раздел "Администрирование"
  2. Выберите "Сотрудники"
  3. Нажмите "Добавить сотрудника"
  4. Заполните информацию о сотруднике

График работы

Как создать смену?

Для создания смены:

  1. Перейдите в раздел "График"
  2. Нажмите "Добавить смену"
  3. Выберите сотрудника, дату и тип смены
  4. Сохраните изменения

Как изменить или отменить смену?

Для изменения смены:

  1. Найдите смену в календаре
  2. Нажмите на смену для просмотра деталей
  3. Выберите "Редактировать" или "Удалить"
  4. Подтвердите изменения

Зарплата

Как рассчитывается зарплата?

Зарплата рассчитывается на основе:

  • Отработанных часов
  • Типа смен (дневные, ночные)
  • Ставки сотрудника
  • Премий и надбавок

Как скачать расчетный лист?

Для скачивания расчетного листа:

  1. Перейдите в раздел "Зарплата"
  2. Выберите нужный период
  3. Нажмите "Скачать расчетный лист"
  4. Файл будет загружен в формате PDF
